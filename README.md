📊 君高财务助手 (Jungao Finance Assistant)

基于 AI 视觉驱动的专业发票报销管理工具

君高财务助手是一款专为财务报销场景设计的轻量化工具。它集成了本地 PDF 解析与 Gemini 2.5 视觉引擎，能够自动识别、汇总并生成符合标准财务规范的 Excel 报销单。


✨ 核心特性

🚀 全格式支持：支持 PDF 电子发票以及 JPG、PNG、TIF、WebP 等多种图片格式。

🧠 AI 视觉识别：采用 Gemini 2.5 视觉引擎，能够深度理解发票排版，精准识别手写体、防伪印章及复杂背景下的金额信息。

⚖️ 大写金额校验：自动捕获“价税合计（大写）”汉字，并将其与小写数字进行逻辑核对，确保报销金额万无一失。

📁 项目自动汇总：支持拖拽文件夹上传。系统将以文件夹名称作为“项目名称”，自动统计张数并汇总金额，极大减少手动输入。

📊 导出专业 Excel：一键生成带边框、合并单元格、合计行及标准签字栏（部门主管、财务审核等）的正式报销单。

🛡️ 隐私安全：核心解析逻辑在浏览器本地运行，不存储任何用户发票数据，确保企业财务信息安全。


🛠️ 如何使用

⚠️ **重要：配置 API Key**
图片格式发票的识别依赖 Google Gemini 视觉引擎。使用前请先在 `index.html` 中找到 `const apiKey = "";`，填入你的 [Google AI Studio API Key](https://aistudio.google.com/apikey) 后方可使用图片识别功能。PDF 发票无需 API Key，可在本地直接解析。

1、访问网页：打开部署好的 GitHub Pages 链接。

2、上传发票：将包含发票的文件夹或多个文件直接拖入页面的“拖拽区域”。

3、数据核对：

  - 观察列表中的“解析状态”，AI 会实时反馈识别进度。

  - 点击金额或发票号码可进行手动微调。

  - 点击“轨迹”可以查看 AI 识别时的原始逻辑，便于纠错。

4、下载报销单：核对无误后，点击“📊 下载正式 Excel 报销单”。


📂 导出效果预览

导出的 Excel 将严格还原标准的纸质报销单排版：

  - 页头：报销单位与开票日期。

  - 主体：按项目汇总的明细行，包含项目日期、说明、张数及金额。

  - 页脚：包含自动生成的中文大写总计金额及规范的四个签字位。


🔧 技术实现

1、前端框架：原生 HTML5 / Tailwind CSS

2、PDF 处理：PDF.js

3、Excel 生成：ExcelJS

4、AI 引擎：Google Gemini API (2.5 Flash Preview)
